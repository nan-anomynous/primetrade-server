# 🚀 PrimeTrade - Full Stack Web Application

PrimeTrade is a full-stack web application built using the MERN/Node.js stack that provides a seamless user experience for authentication and task management with a responsive UI and secure backend API.

---

## ✨ Features

- 🔐 User Authentication (Register / Login)
- 🧑‍💻 Secure API integration
- 📊 Task Management (Create, Read, Update, Delete)
- 🌐 Fully responsive frontend
- ⚡ RESTful API architecture
- ☁️ Deployed backend on Render
- 🌍 Frontend deployed on Vercel

---

## 🛠️ Tech Stack

### Frontend:
- React.js
- JavaScript (ES6+)
- Axios
- CSS / Tailwind (if used)

### Backend:
- Node.js
- Express.js
- REST APIs
- CORS

### Database:
- MongoDB (MongoDB Atlas)

### Deployment:
- Frontend: Vercel
- Backend: Render
